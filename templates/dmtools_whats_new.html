DMTools looks different!

Here's the features that have been added:

- You can now make an exact duplicate of an existing plot
- The "Greatest Hits" feature has been reworked. You can now nominate specific limits to be greatest hits
- The "help" page has been improved. Example and template XML files are available for download from the help page
- An FAQ page has been added
- Inline examples have been added to the "add limit" and "edit limit" pages
- Plots support units of pb, fb, and zb
- Long plot legend labels automatically wrap to a new line instead of being cut off
- Limits are now labeled as "direct". Indirect limits are not supported yet, but will be at some point in the future
- While the aesthetics of the site have changed, the structure of the site is essentially the same
